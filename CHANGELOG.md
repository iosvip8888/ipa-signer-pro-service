# 更新日誌 (Changelog)

所有關於本簽名服務容器鏡像 (Docker Image) 的重大更新都將記錄在此。

[v1.2.5] - 2026-01-08
   修正 (Fixed)

    OpenSSL 跨版本兼容性：解決中央端 OpenSSL 1.0.2k 舊版環境下，因參數語法差異導致的 OCSP 檢測失效（Usage Error）問題。

    指令執行優化：重構 openssl ocsp 調用邏輯，調整參數傳遞順序，顯著提升在舊版 Linux 系統上的兼容性與執行效率。

   新增 (Added)

    國際化 (i18n) 支援：正式導入多語系 UI 框架，支援透過 /static/locales/ 實現語系動態加載與熱切換（包含繁中、簡中、英、越）。

    自動化語系識別：升級 pickLocalizedFields 邏輯，系統可根據瀏覽器語系偏好自動呈現對應內容，優化全球用戶體驗。

[v1.2.4] - 2026-01-07
   優化 (Optimized)

    響應式佈局校準：修正部分行動裝置介面在特定解析度下的顯示異常，確保跨平台視覺一致性。

    UI/UX 體驗升級：全面優化前端視圖細節與動態效果，提升介面整潔度與操作流暢感。

[v1.2.3] - 2025-12-29
   新增 (Added)

    智慧分發模式：於應用管理後台新增「特定分發」開關。支援上傳已簽名安裝包並指定特定 UDID，系統將自動判斷並導流，實現同一站點內的差異化分發。

[v1.2.0] - 2025-12-20
   新增 (Added)

    預訂證書機制 (Hybrid Strategy)：深度優化「企業簽名」與「個人預訂證書」之間的切換邏輯，實現用戶端零感知的穩定轉場。

    OCSP 動態監測：後端引入高性能輪詢引擎，實時監控證書吊銷狀態，將掉簽檢知延遲縮減至最低。

[v1.1.9] - 2025-12-17
   修正 (Fixed)

    內核優化：完成 Docker 容器端核心代碼重構，修復潛在的 Debug 溢位風險，強化執行緒穩定性。

[v1.1.8] - 2025-12-01
   新增 (Added)

    智慧計費邏輯：系統自動辨識「公有池」與「私有池」證書來源。針對私有證書檢索實現零扣費邏輯，精確控管營運成本。

[v1.1.7] - 2025-11-25
   新增 (Added)

    多節點 CDN 接口：整合全局 CDN 加速技術，用戶可自由切換邊緣節點加速安裝或本地分發。後台提供開關選項，靈活適應各種網絡環境。
